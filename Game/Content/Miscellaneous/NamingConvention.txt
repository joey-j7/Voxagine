NAMING:
- All folders and files will be named as descriptive as possible using one or multiple words.
- Words start with a capital letter and are separated using an underscore (_). For example Main_Menu_Background.png.
- Names can only contain characters a-z and 0-9. Other characters (like & and #) are not allowed.
- Names should be as short as possible while maintaining descriptive.
- Folders should be at most 4 layers deep. For example:
		Allowed: 	 Content/Models/House_Models/Roofs/Stone_Roof/Roof.vox
		Not allowed: Content/Models/House_Models/Roofs/Stone_Roofs/Brick_Stone_Roof/Roof.vox
- Files should be stored within a folder that makes sense for the file. For example: House.vox should be stored within the folder Enviourment_Models
- Perforce must be used to make incremental updates to a file. Saving a file with a new version number in the name is not allowed. For example:
		Not allowed: Fishing_Village_Beat1_14.wld -> Fishing_Village_Beat1_15.wld -> Fishing_Village_Beat1_16.wld
- All assets must be on Perforce.

FILES TYPES:
- Textures and images are saved in the .png format. 										For example: Ground_Texture.png
- Models are saved in the .vox format. 														For example: House.vox
- Animations are saved in the .vox format, but contain .anim at the end of the file name. 	For example: Player_Idle.anim.vox
- Audio is saved in the .ogg format. 														For example: Background_Music.ogg
- World/level files are saved in the .wld format. 											For example: Fishing_Village_Beat1.wld
- Prefabs are saved in the .prefab format. 													For example: Temple.prefab

BASE FOLDERS:
There are 10 base folders.
Character_Models:	Contains all character (enemies, player, etc.) models and animations. (Does not contain prefabs!)
Enviourment_Models:	Contains all enviourment (houses, walls, etc.) models. (Does not contain prefabs!)
Models:				Contains all miscellaneous (aim model, pickups, projectiles, etc.) models. (Does not contain prefabs!)
Music:				Contains all music audio.
SFX:				Contains all sound effects audio.
Palettes:			Contains all MagicaVoxel palettes.
Prefabs:			Contains all prefabs.
Textures:			Contains all textures to be used in game. (Does not contain UI textures!)
UI_Art:				Contains all art (textures) for the UI. (Does not contain UI levels!)
Worlds:				Contains all levels, including UI levels.
Miscellaneous:		Contains any assets that do not fit within the above mentioned categories.
